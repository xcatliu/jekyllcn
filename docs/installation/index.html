安装

安装完成 Jekyll 需要几分钟的时间。如果你觉得安装对你来说并不方便, 请 file an issue (或者提交一个 pull request) 来描述一下你的遭遇并告诉我们如何使这个安装过程更加便捷。

事先准备

安装 Jekyll 相当简单，但是你得先做好一些准备工作。开始前你需要确保你在系统里已经有如下配置。

  • Ruby
  • RubyGems
  • Linux, Unix, or Mac OS X
  • NodeJS, 或其他 JavaScript 运行环境
在 Windows 下使用 Jekyll

虽然官方不建议在 Windows 下运行，但你可以看看这个文档： Windows 环境文档

安装 Xcode Command-Line Tools

如果你是Mac用户，你就需要安装 Xcode 和 Command-Line Tools了。下载方式： Preferences → Downloads → Components。

附加功能

根据每个人使用方式的不同，Jekyll 还支持你安装一些附加功能。包括了对 LaTex 的支持，以及使用动态内容渲染引擎。查看 the extras page 获得更多信息。

ProTip™: 允许代码高亮

如果你是一个使用 Jekyll 的程序员，用 Pygments 或者 Rouge 来支持代码高亮吧。当然，使用前请先查看 文档。
